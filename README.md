📊 Integração Python de Análise Financeira

Projeto desenvolvido em Python para análise de ativos financeiros, construção de carteira de investimentos e visualização de indicadores de mercado utilizando dados históricos reais.

O projeto realiza:

coleta automática de dados financeiros
análise de performance da carteira
comparação com benchmarks
cálculo de métricas estatísticas
visualizações interativas
